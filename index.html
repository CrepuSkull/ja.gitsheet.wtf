💻 GitSheet

A dead simple git cheatsheet. Follow @tiivik for updates!

ブランチ
git branch すべてのローカルブランチの一覧を表示する。
git branch -a すべてのリモートブランチとローカルブランチの一覧を表示する。
git checkout -b branch_name ローカルブランチを作成し、そのブランチに切り替える。
git checkout branch_name 既存のローカルブランチに切り替える。
git push origin branch_name 現在のブランチを指定したリモートブランチにプッシュする。
git branch -m new_name 現在のブランチの名前を変更する。
git branch -d branch_name 指定したローカルブランチを削除する。
git push origin :branch_name 指定したリモートブランチを削除する。
ログ
git log --oneline コミット履歴を一行ずつ表示する。
git log -2 最新の2件のコミット履歴を表示する。
git log -p -2 最新の2件のコミット履歴を差分付きで表示する。
git diff 現在の作業ツリーでローカル上のファイルへ加えられた変更をすべて表示する。
git diff file_name 指定したファイルへ加えられた変更の差分を表示する。
git blame file_name 指定したファイル内で誰が何をいつ変更したかを表示する。
git remote show origin ローカルブランチとリモートブランチのマッピングを表示する。
クリーンアップ
git clean -f 追跡されていないファイルをすべて削除する。
git clean -df 追跡されていないファイルとディレクトリをすべて削除する。
git checkout -- . ローカル上のファイルへの変更をすべて元に戻す。
git reset HEAD file_name ステージされたファイルをステージされていない状態に戻す。
タグ
git tag すべてのタグを表示する。
git tag -a tag_name -m "tag message" 新しいタグを作成する。
git push --tags すべてのタグをリモートへプッシュする。
スタッシュ
git stash save "stash_name" && git stash ローカル上のファイルへの変更をスタッシュする。
git stash list すべてのスタッシュの一覧を表示する。
git stash pop 最新のスタッシュを適用し、そのスタッシュを削除する。
